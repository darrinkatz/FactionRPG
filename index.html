FactionRPG

Rules for my diceless, political, player-versus-player RPG.

Download this project as a .zip file Download this project as a tar.gz file

FactionRPG is a political game in which each player controls a powerful faction. Each faction must collaborate with and scheme against the others in order to accomplish their agenda. FactionRPG games require at least five players.
